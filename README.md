# 🍳 Kitchen Buddy - ระบบจัดการคิวครัวอัจฉริยะ

โปรเจกต์ระบบส่งออเดอร์เข้าครัวและคู่มือการทำอาหาร (SOP) เพื่อช่วยให้พนักงานใหม่ทำงานได้อย่างถูกต้องและรวดเร็ว

## 🛠 การติดตั้งเพื่อใช้งาน (Setup)
1. เปิดโปรแกรม Docker Desktop
2. ใช้ VS Code เปิดโฟลเดอร์นี้และรันคำสั่ง:
   ```bash
   docker-compose up -d
   เข้าไปที่ http://localhost:8080 เพื่อใช้งาน phpMyAdmin

นำโค้ดในไฟล์ db_schema.sql ไปรันในหน้า SQL ของ phpMyAdmin เพื่อสร้างตาราง
